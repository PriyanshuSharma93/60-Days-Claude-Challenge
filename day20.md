# 🚀 Day 20 – Claude AI Mastery Challenge

## 📅 Challenge

**ABTalks – 60 Days Claude AI Mastery Challenge**

**Phase 3:** AI-Powered Interactive Web Applications

**Topic:** AI Face Puzzle Game

---

# 🎯 Objective

The objective of Day 20 was to build a **fully functional browser-based Face Puzzle Game** using Claude AI and Prompt Engineering.

The challenge focused on generating a complete, self-contained HTML application that integrates the user's webcam, captures a live photo, converts it into a puzzle, and provides an interactive gaming experience with modern UI, drag-and-drop mechanics, touch support, and persistent score tracking.

---

# ✅ Tasks Completed

* Generated a complete single-file HTML application
* Integrated webcam access using `getUserMedia()`
* Captured user photos with Canvas API
* Implemented puzzle generation from captured images
* Added multiple difficulty levels (3×3, 4×4, 5×5)
* Enabled drag-and-drop interactions
* Added touch gesture support for mobile devices
* Implemented timer and move counter
* Tracked correctly placed puzzle pieces
* Added automatic win detection
* Built a leaderboard using Local Storage
* Created responsive UI for desktop and mobile
* Tested gameplay across different difficulty levels

---

# 📚 What I Learned

## 1. Browser Camera Integration

The project uses the **MediaDevices API** to access the user's webcam directly from the browser.

Features include:

* Live Camera Preview
* Front Camera Access
* Photo Capture
* Permission Handling
* Retake Photo Option

This demonstrated how modern browser APIs enable interactive applications without external software.

---

## 2. Dynamic Puzzle Generation

After capturing the image, the application automatically:

* Divides the photo into equal tiles
* Creates puzzle pieces based on selected difficulty
* Randomly shuffles the pieces
* Maintains a playable puzzle layout

Supported puzzle sizes include:

* 3×3
* 4×4
* 5×5

---

## 3. Interactive Gameplay

The puzzle supports smooth interactions on both desktop and mobile devices.

Gameplay features include:

* Drag & Drop
* Touch Gestures
* Piece Swapping
* Snap-to-Grid
* Active Drag Highlighting
* Correct Position Indicators

These interactions improve the overall gaming experience while maintaining responsive performance.

---

## 4. Game Progress Tracking

The application continuously tracks player progress by displaying:

* Live Timer
* Total Moves
* Correctly Placed Pieces
* Completion Status

When the puzzle is solved, the game automatically detects completion and displays the final statistics.

---

## 5. Persistent Leaderboard

The project stores player records locally using the browser's **Local Storage**.

Each record includes:

* Completion Date
* Final Time
* Number of Moves
* Puzzle Difficulty

The Top 5 best performances are preserved across browser sessions.

---

## 6. Responsive UI Design

The application was designed with a modern interface featuring:

* Responsive Layout
* Mobile-Friendly Controls
* Modern Card-Based Design
* Smooth Animations
* Interactive Buttons
* Clean User Experience

Everything is contained within a **single HTML file** with inline CSS and JavaScript.
