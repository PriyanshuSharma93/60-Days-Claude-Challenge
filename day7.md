# 📚 What I Learned

## 1. Why Different Claude Models Exist

Not every task requires the most powerful AI model.

Claude provides different models optimized for different types of work:

* **Haiku** → Speed and efficiency
* **Sonnet** → Balanced performance for everyday tasks
* **Opus** → Advanced reasoning and complex problem-solving

Choosing the right model saves both time and resources while improving productivity.

---

## 2. Understanding Each Claude Model

### ⚡ Haiku

Best suited for:

* Quick questions
* Summarizing notes
* Brainstorming ideas
* Grammar correction
* Simple coding queries
* Everyday productivity tasks

**Advantages**

* Fast responses
* Low latency
* Great for repetitive work

---

### 🧠 Sonnet

Best suited for:

* Java programming
* Spring Boot development
* DSA problem solving
* Debugging code
* Learning new concepts
* Writing technical documentation
* Project planning

**Advantages**

* Balanced speed and reasoning
* High-quality coding assistance
* Ideal for daily development work

---

### 🚀 Opus

Best suited for:

* Complex software architecture
* Deep technical research
* System design
* Multi-step reasoning
* Business strategy
* Long-form content generation
* Advanced AI workflows

**Advantages**

* Strongest reasoning capability
* Excellent for difficult and detailed tasks

---

# 3. Understanding Effort Levels

Claude allows different effort settings depending on how much reasoning is required.

### 🟢 Low

Use for:

* Simple questions
* Summaries
* Basic explanations
* Quick productivity tasks

---

### 🔵 Standard

Use for:

* Daily coding
* Learning
* Documentation
* Resume improvements
* LeetCode explanations
* General development tasks

---

### 🟠 High

Use for:

* Debugging difficult issues
* Software architecture
* Advanced project planning
* Technical interviews
* Design discussions

---

### 🔴 Max

Use for:

* Deep research
* Complex reasoning
* Large project planning
* Multi-step workflows
* High-level decision making

# 💡 Biggest Mistakes to Avoid

* Using the same model for every task.
* Choosing High or Max effort for simple questions.
* Expecting short prompts to produce detailed outputs.
* Ignoring context when asking AI questions.
* Relying on AI without reviewing important outputs.

# 🧠 Key Takeaways

* Different Claude models are designed for different types of work.
* Selecting the right model improves both speed and output quality.
* Effort settings should match the complexity of the task.

* High and Max effort should be reserved for complex reasoning tasks.
