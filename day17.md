# 🚀 Day 17 – Claude AI Mastery Challenge

## 📅 Challenge

**ABTalks – 60 Days Claude AI Mastery Challenge**

**Phase 3:** AI-Powered Data Analytics & Dashboard Development

**Topic:** Vehicle Fuel Cost & Sustainability Dashboard

---

# 🎯 Objective

The objective of Day 17 was to build an **AI-powered interactive dashboard** that reads vehicle data from a CSV file, computes key performance metrics, and visualizes the results in a responsive HTML dashboard.

For this project, I used my **Land Rover Defender 110 (Diesel)** as the reference vehicle to analyze running costs, maintenance expenses, fuel efficiency, CO₂ emissions, and compare different fuel types using real data.

The challenge focused on combining **data analytics**, **prompt engineering**, and **frontend dashboard development** into a single AI-powered workflow.

---

# 🚙 Vehicle Details

* **Vehicle:** Land Rover Defender 110
* **Fuel Type:** Diesel
* **Usage:** Mixed
* **Average Distance:** 1,000 km/month
* **Vehicle Age:** 3 Years

---

# ✅ Tasks Completed

* Read and processed vehicle data from a CSV file
* Calculated Cost per Kilometer
* Calculated CO₂ Emissions per Kilometer
* Calculated Maintenance Cost per Kilometer
* Calculated Average Refuel/Recharging Time
* Categorized vehicles using Age Buckets
* Compared Diesel, Petrol, E85, CNG, and EV
* Calculated E85 Fuel Savings & Break-even Price
* Generated an E85 Performance Score
* Built a fully responsive HTML dashboard
* Created interactive SVG-based charts without external libraries

---

# 📚 What I Learned

## 1. Data Processing with AI

Claude AI transformed raw CSV data into meaningful business insights by automatically calculating:

* Cost per Kilometer
* CO₂ Emissions
* Maintenance Cost
* Monthly Fuel Cost
* Refueling Time
* Vehicle Age Analysis

This demonstrated how AI can automate repetitive analytical tasks efficiently.

---

## 2. KPI Dashboard Development

The dashboard highlighted important metrics through KPI cards, including:

* Current Fuel Cost per KM
* E85 Cost Comparison
* Monthly Fuel Expense
* Break-even Fuel Price
* Fuel Efficiency Metrics

These KPIs provide a quick overview of vehicle performance and operating costs.

---

## 3. Interactive Data Visualization

The dashboard featured custom SVG-based visualizations, including:

* Bar Chart (Fuel Cost Comparison)
* Doughnut Chart (CO₂ Emissions)
* Line Chart (Cost vs Vehicle Age)
* Animated SVG Gauge (E85 Score)
* Fuel Comparison Cards

Building these visualizations without external libraries improved my understanding of frontend rendering techniques.

---

## 4. Responsive Dashboard Design

The application included:

* Dark Navy Theme
* Glassmorphism UI
* Mobile-Responsive Layout
* Interactive Hover Effects
* Pure HTML, CSS & JavaScript
* SVG Graphics (No CDN Required)

This reinforced modern dashboard design principles while maintaining performance.

---

## 5. Practical Data Analytics

Using my **Land Rover Defender 110** as the case study made the project more practical by helping analyze:

* Running Costs
* Maintenance Expenses
* Fuel Comparisons
* Environmental Impact
* Vehicle Ownership Metrics

This challenge demonstrated how AI can convert raw datasets into actionable insights through interactive dashboards.
